# ContractsCompare Precedent Library

Public M&A agreement clauses extracted from SEC EDGAR 
filings. Used to ground AI suggestions in real 
negotiated outcomes.

## Deal criteria
- Deal size: $250M - $750M
- Years: 2019-2024
- Industries: Technology, Healthcare, Financial Services,
  Real Estate, Manufacturing, Consumer, Energy, Media
- Source: SEC EDGAR public filings

## Clause types extracted
1. Material Adverse Effect (MAE) definition
2. Representations & Warranties - Business/Operations
3. Indemnification - cap and basket
4. Closing conditions
5. Termination rights and fees
6. Non-compete / non-solicit
7. Purchase price adjustments
